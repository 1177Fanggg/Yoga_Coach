## 設計要點

### 色彩配置
- **主色調**: 深紫色系 (#6366f1, #8b5cf6)  //黑百配 不要太花俏
- **成功色**: 綠色 (#10b981)
- **警告色**: 黃色 (#f59e0b)
- **錯誤色**: 紅色 (#ef4444)
- **背景**: 深色主題 (#1a1a2e, #16213e)
- **文字**: 白色 (#ffffff) / 淺灰 (#94a3b8)
- 不要icon
### 設計風格
- 現代化玻璃態 (Glassmorphism)
- 柔和漸層背景
- 微動畫 (hover effects)
- 圓角卡片設計
- 清晰的視覺層次

### 響應式設計
- 桌面: 左右分欄布局
- 平板: 堆疊布局
- 手機: 單欄全寬

### 字體
- Google Fonts: Inter (主要字體)
- 等寬字體用於數據顯示

---

## 技術實作要點

1. **即時練習頁面**
   - 使用 MediaPipe 前端版本處理相機畫面
   - WebSocket 接收即時回饋
   - Canvas 疊加骨架繪製

2. **歷史記錄頁面**
   - 分頁或無限滾動載入
   - 本地排序與篩選

3. **Session 詳情頁面**
   - Modal 或獨立頁面
   - HTML5 video 播放器
   - 可折疊的姿勢詳情卡片
